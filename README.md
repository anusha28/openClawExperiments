A personal OpenClaw workspace for learning and experimenting with agent workflows, memory files, and reusable skills.

## What this repo is

This repository captures the **shareable parts** of an OpenClaw workspace:
- workspace guidance and persona files
- reusable skills
- lightweight project docs
- repo-safe configuration and examples

## What’s included

### Workspace files
- `AGENTS.md` — workspace operating rules
- `SOUL.md` — assistant voice/persona guidance
- `USER.md` — user profile notes
- `IDENTITY.md` — assistant identity record
- `TOOLS.md` — local tool notes template
- `HEARTBEAT.md` — heartbeat task file

### Skills
- `skills/document-summary/` — summarize documents
- `skills/email-triage/` — compact inbox triage workflow
- `skills/imap-smtp-email/` — IMAP/SMTP email tooling and setup scripts
- `skills/quick-note/` — quick note workflow
